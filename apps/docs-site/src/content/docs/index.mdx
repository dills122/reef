Reef models the lifecycle of privately matched institutional trades — order intake, hidden-book matching, executions, and the post-trade path toward settlement — with production-shaped architecture that stays easy to run and inspect locally. A tournament-style **Bot Arena** is under active design on top of the same venue command paths.
