# Post-Match Reference Notes (May 2026)

## Purpose

Capture source-backed reference points used to calibrate Reef post-match standards.

## Sources and key takeaways

1. SEC T+1 and Rule 15c6-2 operating expectations  
Source: [SEC T+1 FAQ](https://www.sec.gov/exams/educationhelpguidesfaqs/t1-faq)
- U.S. standard settlement cycle is T+1.
- Rule 15c6-2 requires steps to complete allocation/confirmation/affirmation as soon as technologically practicable and no later than end of trade date.

2. FINRA implementation alignment for T+1  
Source: [FINRA final reminder (May 22, 2024)](https://www.finra.org/filing-reporting/technical-notice/final-reminder-t-1-settlement-052224)
- FINRA reporting/process changes aligned with the May 28, 2024 T+1 transition.

3. NSCC central counterparty and netting role  
Sources:
- [NSCC overview](https://www.dtcc.com/about/businesses-and-subsidiaries/nscc)
- [CNS overview](https://www.dtcc.com/clearing-and-settlement-services/equities-clearing-services/cns)
- NSCC acts as central counterparty for covered flows.
- CNS nets positions by security/member and supports fail-control behaviors.

4. DTC/NSCC settlement path and DvP context  
Source: [DTCC settlement overview](https://www.dtcc.com/understanding-settlement/index.html)
- Clearing and settlement are distinct operational layers.
- Settlement processing includes delivery/payment mechanics and risk controls.

## How these references map to Reef standards

- Default calendar/cycle profile: U.S. T+1
- Explicit compare/affirmation stage with deadlines
- Explicit clearing and novation stage before netting
- Explicit netting outputs before settlement obligation intake
- Traceable event model across every stage for operations and learning
